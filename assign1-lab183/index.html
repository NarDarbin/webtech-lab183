William Shakespeare The Chandos Portrait of William Shakespeare English playwright, poet and actor This was long thought to be the only portrait of William Shakespeare that had any claim to have been painted from life, until another possible life portrait, the Cobbe portrait, was revealed in 2009. The portrait is known as the 'Chandos portrait' after a previous owner, James Brydges, 1st Duke of Chandos. It was the first portrait to be acquired by the National Portrait Gallery in 1856. The artist may be by a painter called John Taylor who was an important member of the Painter-Stainers' Company.

Compatibillity Report

All the differences in report hereby were currently corrected using the solution provided (if the solution is provided)

Desktop Mode

Hereby are presented the website page opened in Google Chrome (on the right) and Mozilla Firefox (on the left) browsers. It's important to note that the page produces no errors in both of the browers

Padding

Differences: the padding in the Firefox seem to be a bit bigger in the navigation bar reflected by its height. This is the result of one page being wider than the other as this does not change when resizing windows to the reasonable extent

Solution: adjusting the Chrome margin to be corresponding to Firefox inside of the @supports (-moz-transform: translate(0, 0)) media query

Font Weight

Differences: Chrome and Firefox treat font weight differently with Firefox making those bolder than Chrome.

Solution: adjust the weight to the desirable on Firefox browser and then using media query @supports (-moz-transform: translate(0, 0)) adjust the weight on Chrome browser to the desirable to make it bolder.

Mobile Version

Mobile versions were tested using a browser emulator. Browser differences were the same as on desktop version; however, the font weight difference in paragraphs became more apparent on mobile version. The following picture below demostrated how the website compares when viewed on iPhone 12 Pro.

Differences: Chrome and Firefox treat font weight differently with Firefox making those bolder than Chrome and font size in Firefox seem bigger.

Solution: adjust the weight to the desirable on Firefox browser and then using media query @supports (-moz-transform: translate(0, 0)) adjust the weight on Chrome browser to the desirable to make it bolder.

Print Media Version

Printed versions were tested on differences in the printing emulator as shown below. Almost all the differences versions have are inherit to ones desktop and mobile versions had.

Default Header

Differences: Firefox by default shows the title of the website and its address at the top of the page printed while Google leaves the space blank.
